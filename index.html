Panduan Penggunaan

PDF Tools

  1. Compress: drag PDF, pilih mode, atur kualitas, klik Kompres Semua
  2. Gabung: upload beberapa PDF, atur urutan dengan drag, klik Gabung
  3. Edit Halaman: upload PDF, pilih operasi (hapus/ambil/sisipkan/putar), proses
  4. Split: upload PDF, pilih mode pemisahan, preview hasil, unduh ZIP
  5. Batch Combine: upload template Header/Footer, upload file, proses semua

Batch File Rename

  1. Pilih folder berisi file yang akan direname
  2. Export daftar file sebagai CSV untuk referensi
  3. Paste daftar nama baru (satu per baris, tanpa ekstensi)
  4. Cek Preview sebelum eksekusi
  5. Klik Eksekusi Rename

File Organizer

  1. Pilih folder sumber
  2. Buat rules: kata kunci di nama file kemudian pindah ke folder apa
  3. Import rules dari CSV jika sudah punya daftar
  4. Cek Preview hasil pemetaan
  5. Klik Eksekusi untuk memindahkan file
PDF Tools bekerja di semua browser modern. Batch Rename dan File Organizer membutuhkan Chrome 86 ke atas atau Edge 86 ke atas karena menggunakan File System Access API. Firefox dan Safari belum mendukung fitur akses folder secara langsung.
ENV Working Tools by Ifan
Kumpulan tools kerja: PDF Tools, Batch Rename, dan File Organizer. Berjalan sepenuhnya di browser. Aman, gratis, tanpa upload ke server.

Aplikasi ini lahir dari kebutuhan pekerjaan. Banyak tools PDF online mengharuskan upload dokumen ke server pihak ketiga. Ini menimbulkan risiko kebocoran data, terutama untuk dokumen teknis atau confidential.

ENV Working Tools berjalan sepenuhnya di browser. File tidak pernah meninggalkan perangkatmu. Tidak perlu login, tidak ada iklan, tidak ada biaya berlangganan.

Source code terbuka dan bisa diperiksa langsung di file HTML ini. Jika ada saran atau bug, silakan kirim lewat form di bawah.
